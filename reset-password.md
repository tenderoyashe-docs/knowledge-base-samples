# How to Reset Your Password

## Summary
Use this guide if you cannot log in because you forgot your password.

## Steps
1. Go to the login page.
2. Select **Forgot password**.
3. Enter your registered email address.
4. Select **Submit**.
5. Open the password reset email and select the reset link.
6. Create a new password and confirm.

## Notes
- The reset link expires after 30 minutes.
- If you do not receive the email, check your spam folder.
- If the issue continues, contact Support.
